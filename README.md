Проектная работа 6 "Умная таблица.
Задача — обеспечить получение данных с сервера заменяя предыдущий статичный вид получения данных на API(сервер). Сделать это нужно так, чтобы передача происходила через квери параметров поиска, фильтрации, сортировки и пагинации. В компонентах мы заменим прямую работу с данными на работу с состоянием запроса — то есть набором параметров, которые используются для запроса данных с сервера. При этом функция рендера станет асинхронной и сначала будет получать данные с сервера по заданным критериям, а уже после — выводить их как раньше.
